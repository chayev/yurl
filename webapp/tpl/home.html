yURL: Deep Linking Validator

iOS Universal Links and Android App Links

yURL allows you to validate whether a URL is properly configured for Universal Links and App Links. This allows you to check if the apple-app-site-association and assetlinks.json on a website is properly formatted for deep links as defined by Apple and Google.

Enter a URL and select whether you want to validate on iOS or Android to see the results.
