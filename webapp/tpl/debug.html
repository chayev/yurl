What is yURL

yURL allows you to validate whether a URL is properly configured for Universal Links and App Links.

Why use it

yURL verifies if the assetlinks.json and apple-app-site-association are properly formatted for deep links as defined by Google and Apple.

Valid deep links enable users to be routed to specific app content. This drives better user experience, better user engagement and higher conversion rates.
